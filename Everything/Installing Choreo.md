# Installing Choreo

So, you're here to install Choreo, congrats. It's very addicting and satisfying to make a good auto!

## Linux
Noice, very, very good choice.

Granted, you probably know how to install applications on your own distro, so I will just point you to the file and you can install it from there: https://github.com/SleipnirGroup/Choreo/releases/

Make sure to check the sha! And also install the correct architecture (they have support for a lot of stuff, which is great, but sometimes a bit confusing).

## Mac
Another very respectable operating system (don't tell Elliot).

Releases page: https://github.com/SleipnirGroup/Choreo/releases/

For installing the actual app (dmg — you own the computer / your user can install into /Applications): use Choreo-v20xx.x.x-macOS-aarch64.dmg

If you cannot install that because you are not admin of the computer, then use the standalone version: Choreo-v20xx.x.x-macOS-aarch64-standalone.zip

## Windows
Uh welp... Windows huh? Ya know, almost all Windows computers are x86_64, so you could just... uninstall Windows and run Linux instead.

Well anyways, I guess I'll allow it. Here's the release page (same as all of the others): https://github.com/SleipnirGroup/Choreo/releases/

For the actual app install (you have rights to write to the Applications folder): Choreo-v20xx.x.x-Windows-aarch64-setup.exe

For the standalone version (where you may not have full rights to the computer): use Choreo-v20xx.x.x-Windows-aarch64-standalone.zip
