# Enable and disable the robot

Enabling and disabling the robot is very simple, you just have to be able to do it safely.

## How to do it
1. **Check the controllers** — Check that any connected controllers are not sitting face down on the table. If they are, then when you click enable, the robot will shoot in whichever direction the joysticks are being pushed down (not very good...).
2. **Set the mode to TeleOperated** — Make sure that the right most box above the Enable/Disable buttons is in "TeleOperated" mode. If it is in "Autonomous", then if an auto is selected, it will run the auto, which can be bad if the field is not set up for running an auto at that moment.
3. **Clear the field, then enable** — Make sure the field is clear and make sure that everyone on the field is aware the robot is being enabled. When you are about to enable the robot, yell "Enabling" and then click the green "Enable" button. You will know it is enabled when the RSL (Robot Signal Light, aka the orange light) begins blinking.

## Troubleshooting
**The enable button won't do anything:** Check that the robot is connected. You will know it's connected if the battery icon is green and updating and the "Communication" status light (one of the three labeled Communication, Robot Code and Joysticks) is green. If it is connected, verify that the robot is not in E-Stop. You will know it's in E-Stop if, when you click Enable, the Driver Station window turns orange and it says "E-STOP". In that case you will have to go over to the robot and restart it by turning it off and then turning it back on again :)
