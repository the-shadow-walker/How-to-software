# Mac
Another very respectable operating system (don't tell Elliot)

Releases page: https://github.com/SleipnirGroup/Choreo/releases/

For installing the actual app (dmg, you own the computer/your user can install into the /Application): Use Choreo-v20xx.x.x-macOS-aarch64.dmg

If you cannot install that because you are not admin of the computer then use the stand alone version: Choreo-v20xx.x.x-macOS-aarch64-standalone.zip
