So, you've (hopefully) opened chore now and are at a screen that looks something like this (but it will of course vary from game field to game field):
![[Pasted image 20260612235150.png]]

To set it up so that all of your autos save to our code, click the folder icon in the top left 
![[Pasted image 20260613000314.png]]

It should open your file explorer, navigate to the spot where the robot code is saved under and go to ./src/main/deploy/choreo. There, you will see any .traj files (already built paths) that have been made.

You now have the project configured!
